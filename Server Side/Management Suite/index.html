Globlock: 2 Factor Document Access Control, Repository and File Tokenization Show me!

Welcome to Globlock

This is Globlock, a fully extensible software system and API designed by Alex Quigley and developed as part of a 4th year project for 'BSHCE Mobile and Networking' honors degree with National College of Ireland.

File Management?

Globlock provides a document management system and file repository with 2 factor authentication, using open source hardware and software. A high level user can manage and control document revisions and access by grouping them into individual projects called 'Globes'.

Globes you say?

Each Globe takes the abstract physical form of a Globe Object with an embedded and unique RFID. This allows a physical dimension and tokenisation of documents and files, and without physical possession of the Globes, the files cannot be accessed.

How?

The entire system uses a robust central hand written and tested API, which supports this document tokenization and masks its underlying architecture from the user to provide a secure and streamlined file management access and repository solution.
